## 🌍 New Tab Dashboard

크롬 새 탭에서 사용할 수 있는 **개인 맞춤형 대시보드**입니다.
실시간 시계, 랜덤 배경화면, 명언, 검색 기능, 북마크 기능을 제공합니다.

## 📌 Features

- 실시간 시계 표시 & 날짜 표시
- 랜덤 배경 이미지
- 랜덤 명언 표시(하루 한번)
- 검색 기능(Google)
- 반응형 레이아웃

---

## 🚀 How To Use

1. 해당 프로젝트를 다운로드 받는다.
2. 크롬 주소창에 [크롬 확장 프로그램](chrome://extensions)을 입력한다.
3. 압축해제된 확장 프로그램 로드를 선택한다.
4. 다운로드 받은 폴더를 선택한다.
